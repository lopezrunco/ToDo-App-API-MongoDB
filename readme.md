# API REST con el framework Express (MongoDB)

## Instrucciones de uso

Crear el archivo .env con variables de entorno

```bash
API_PORT=3000
JWT_KEY=super_secret_key
DB_HOST=localhost
DB_PORT=3306
DB_NAME=todos
DB_USER=root
DB_PASSWORD=root
```

```bash
# Instalar dependencias
npm i

# Insertar datos de prueba
npm run seed

# Iniciar API
npm start
```
